Transform any WordPress website into a full-featured Progressive Web App with comprehensive OneSignal push notification integration.

== Description ==

OneSignal PWA for WordPress is the definitive solution for converting WordPress sites into installable, offline-capable Progressive Web Apps with enterprise-grade push notification capabilities.

= Key Features =

* **PWA Capabilities**
  * Automatic manifest.json generation
  * Service worker with advanced caching strategies
  * Offline functionality
  * Installable app experience
  * App shortcuts
  * Custom offline pages

* **OneSignal Integration**
  * Complete REST API integration
  * Web SDK implementation
  * Push notifications
  * User segmentation
  * Workflow automation
  * Analytics and tracking

* **WooCommerce Integration**
  * Order notifications
  * Cart abandonment recovery
  * Back in stock alerts
  * Price drop notifications
  * Review requests

* **Analytics Dashboard**
  * Subscriber tracking
  * Notification performance
  * Conversion tracking
  * Device and browser breakdowns
  * Geographic insights

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/onesignal-pwa/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Settings â†’ OneSignal PWA
4. Enter your OneSignal App ID and API keys
5. Configure your PWA settings
6. Upload your app icon
7. Start sending notifications!

== Frequently Asked Questions ==

= Do I need an OneSignal account? =

Yes, you need a free or paid OneSignal account. Visit https://onesignal.com to sign up.

= Does this work with WooCommerce? =

Yes! The plugin includes comprehensive WooCommerce integration for cart abandonment, order notifications, and more.

= Is HTTPS required? =

Yes, Progressive Web Apps and push notifications require HTTPS to function properly.

= Can I customize the PWA appearance? =

Absolutely! You can customize the app name, colors, icons, and more from the PWA Settings page.
